# なにこれ

[syakoo](https://github.com/syakoo)
によって作られた、自作のSVGやReact Componentsを保管する図書館。
自分用に作ったけれど、誰でも使って構いません。その代わりクオリティは保証しません。
OSSなので、追加のAssetsやバグの報告、プルリクエストは歓迎しています。


使う際は、[このリポジトリ](https://github.com/syakoo/my-assets-lib)
にスターをください。喜びます。
